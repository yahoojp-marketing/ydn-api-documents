# AdGroupTarget
AdGroupTargetオブジェクトは、広告グループに設定されているターゲティング情報を格納します。<br>
TC-CI-00000073は「その他」の地域コードです。DictionaryServiceのgetGeographicLocationでは取得できず、地域ターゲティグでは指定できません。
