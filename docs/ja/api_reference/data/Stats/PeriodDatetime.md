# PeriodDatetime
PeriodDatetimeオブジェクトは、統計情報の処理を開始/完了した日時です。

### Service

### Namespace
[StatsService#Namespace](../../services/StatsService.md#namespace)

|---|---|---|---|---|---|---|
| periodDate | String | 日<br>※フォーマット：yyyyMMdd | yes | - | - | - |
| periodTime | String | 時分<br>※フォーマット：HHmmss | yes | - | - | - |
