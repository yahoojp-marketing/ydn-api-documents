# UnknownDomainFlg (enum)
UnknownDomainFlgは、不明ドメインフラグを表します。
### Service
+ [PlacementUrlListService](../services/PlacementUrlListService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| TRUE| string| 不明ドメインを含む |
| FALSE| string| 不明ドメインを含まない |
