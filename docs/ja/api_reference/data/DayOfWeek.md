# DayOfWeek (enum)
DayOfWeekは、配信する曜日を表します。
### Service
+ [AdGroupTargetService](../services/AdGroupTargetService.md)

| 値 | 説明 | 
|---|---|
| MONDAY| 月曜日です。 |
| TUESDAY| 火曜日です。 |
| WEDNESDAY| 水曜日です。 |
| THURSDAY| 木曜日です。 |
| FRIDAY| 金曜日です。 |
| SATURDAY| 土曜日です。 |
| SUNDAY| 日曜日です。 |
