# IsPreset (enum)
IsPresetは、過去の訪問履歴の対象の有無を表します。
### Service
+ [RetargetingTagService](../services/RetargetingTagService.md)

| 値 | 説明 | 
|---|---|
| TRUE| 過去の訪問履歴を含みます。 |
| FALSE| 過去の訪問履歴を含みません。 |
