# ConversionOptimizerEligibilityFlg (enum)
ConversionOptimizerEligibilityFlgは、コンバージョン最適化機能が使用可能であるか表示します。
### Service
+ [CampaignService](../services/CampaignService.md)

| 値 | 説明 | 
|---|---|
| ENABLE| コンバージョン最適化機能は使用可能です。 |
| DISABLE| コンバージョン最適化機能は使用できません。 |
