# ReportIntervalType (enum)
ReportIntervalTypeは定期レポート作成タイミングを表します。
### Service
+ [ReportDefinitionService](../services/ReportDefinitionService.md)

| 値 | 説明 | 
|---|---|
| ONETIME| 一度のみ(Default) |
| DAILY| 毎日 |
| EVERYSUN| 毎週日曜 |
| EVERYMON| 毎週月曜 |
| EVERYTUE| 毎週火曜 |
| EVERYWED| 毎週水曜 |
| EVERYTHU| 毎週木曜 |
| EVERYFRI| 毎週金曜 |
| EVERYSAT| 毎週土曜 |
| SPECIFYDAY| 指定日(毎月実行)。別フィールドにて日にちを指定。 |
