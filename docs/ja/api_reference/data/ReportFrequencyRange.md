# ReportFrequencyRange (enum)
ReportFrequencyRangeは、フリークエンシーの計測期間を表します。
### Service
+ [ReportDefinitionService](../services/ReportDefinitionService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| DAILY| string | 1日単位です。 |
| WEEKLY| string | 1週間単位です。 |
| MONTHLY| string | 1ヶ月単位です。 |
