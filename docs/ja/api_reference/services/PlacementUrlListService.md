# PlacementUrlListService
PlacementUrlListServiceは、プレイスメントURLリストの作成・更新・削除・取得を行います。
#### WSDL
| environment | url |
|---|---|
| production  | https://location.im.yahooapis.jp/services/Vx.x/PlacementUrlListService?wsdl|
| sandbox  | https://sandbox.im.yahooapis.jp/services/Vx.x/PlacementUrlListService?wsdl|
#### Namespace
http://im.yahooapis.jp/V5
#### サービス概要
プレイスメントURLリストの作成・更新・削除・取得を行います。
#### 操作
PlacementUrlListServiceで提供される操作を説明します。
