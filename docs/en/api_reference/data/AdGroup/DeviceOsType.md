

# DeviceOsType (enum)

DeviceOsType serves OS types for devices to be posted on.

#### Service

+ [AdGroupService](../../services/AdGroupService.md)

#### Namespace

[AdGroupService#Namespace](../../services/AdGroupService.md#namespace)

| Enumeration  |       Type       |          Description          |
| ------------ | ---------------- | ----------------------------- |
| IOS | xsd:string | Deliver to iOS. |
| ANDROID | xsd:string | Deliver to Android. |
| NONE | xsd:string | Deliver to all OS.<br/>*Setting NONE is only available for AdGroupService. |
